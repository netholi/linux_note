- **Check Motherboard/Baseboard Info:**
 ``` bash
 sudo dmidecode -t baseboard
 ```
   
- **Check Physical Memory (RAM) details** (e.g., speed, manufacturer, form factor):
``` bash
sudo dmidecode -t memory
```
   
- **Check maximum supported memory** for your motherboard:
 ``` bash
   sudo dmidecode -t 16
 ```
 
- **Check Processor/CPU information:**
``` bash
 sudo dmidecode -t processor
```
