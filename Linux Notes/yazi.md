| Key            | Action                                          | Notes                                             |
| -------------- | ----------------------------------------------- | ------------------------------------------------- |
| `~`            | Show help                                       |                                                   |
| `gg`           | Go to the top                                   |                                                   |
| `G`            | Go to the bottom                                |                                                   |
| `Z`            | Jump to a previously visited directory          | Requires the **zoxide** application               |
| `z`            | Search using **fzf**                            |                                                   |
| `H`            | Move to the previous directory                  | Directory history                                 |
| `L`            | Move to the next directory                      | Directory history                                 |
| `o` or `Enter` | Open a file                                     |                                                   |
| `Space`        | Select multiple files                           |                                                   |
| `y`            | Copy selected files                             |                                                   |
| `p`            | Paste copied/cut files                          |                                                   |
| `x`            | Cut selected files                              |                                                   |
| `d`            | Delete selected file(s)                         | Moves to trash (if supported)                     |
| `D`            | Permanently delete selected file(s)             |                                                   |
| `a`            | Create a file or folder                         | Add `/` at the end of the name to create a folder |
| `r`            | Rename a file                                   |                                                   |
| `Tab`          | View file information                           |                                                   |
| `Ctrl + A`     | Select all files                                |                                                   |
| `.`            | Show/Hide hidden files                          |                                                   |
| `;`            | Run a shell command                             |                                                   |
| `c`            | Copy file path, file name, directory path, etc. |                                                   |
| `f`            | Filter files                                    |                                                   |
| `/` or `?`     | Search                                          | Press `n` to jump to the next match               |
| `s`            | Search files using the `fdfind` command         |                                                   |
| `S`            | Search within a file                            | Press `Esc` or `Ctrl + S` to cancel               |
| `,`            | Sort files                                      | Displays available sort keys                      |
| `m`            | Toggle line mode                                | Shows file permissions, ownership, etc.           |
| `g`            | Go to a frequently used folder                  |                                                   |
| `t`            | Create a new tab                                |                                                   |
| `[` / `]`      | Switch between tabs                             | You can also use the number keys                  |
| `{` / `}`      | Swap tab positions                              |                                                   |
| `Ctrl + C`     | Close the current tab                           |                                                   |
