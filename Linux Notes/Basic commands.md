
>[!info] whoami 
>Displays the current username.
>whoami will return your current logged-in username.

>[!info] pwd
 Prints the current working directory.

   >[!info] nano
   >A simple command-line text editor.
   >Example: nano /etc/hosts opens the hosts file for editing.
   
>[!info]  which 
>Locates the executable file for a command.
>Example: which [command] shows the full path (e.g., /usr/sbin/ip) of that command.   

 >[!info] ls
 > Lists files in a directory. Used frequently to inspect default configurations or home directories.
``` bash
ls -la /etc/skel 
```

>[!info] tail 
>Displays the last part of a file (commonly used here to view /etc/group).
``` bash
 tail /etc/group 
```

>[!info] touch
>Creates an empty file.
``` bash
 touch file1.txt 
```

>[!info] mkdir 
>Used to create a new directory.
``` bash
 mkdir test_files 
```

>[!info] less 
>Used to view the contents of a file one page at a time.
``` bash
 less .bashrc 
```
