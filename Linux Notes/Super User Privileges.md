

>[!info] userdel 
Deletes a user account. use -r to remove home dir
>userdel [username] removes the specified user from the system.

>[!info] su
>su Stands for "substitute user," used to switch to another user account.
>su - [username] starts a login session for that user.
>Typing exit returns you to your previous user session.

>[!info] sudo
>"Super user do," allows you to execute a command as the super user.
>sudo [command] runs the specified command with administrative privileges.

>[!info] visudo 
>The recommended tool for editing the /etc/sudoers file, as it checks for syntax errors before saving.
> visudo launches the configuration file editor safely.

>[!info] sudoedit 
> An alias used to edit a file using your system's default editor with sudo privileges.
>  Example: sudoedit /etc/hosts opens the hosts file in your preferred editor as root.
